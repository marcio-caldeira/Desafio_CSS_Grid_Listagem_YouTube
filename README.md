# 🎬 Clone da Página de Vídeos do YouTube (Grid Layout)

Este projeto consiste na construção de uma **página de listagem de vídeos inspirada no YouTube**, utilizando **HTML e CSS**, com foco principal na aplicação de **CSS Grid Layout** e conceitos de responsividade.

---

## 🚀 Objetivo do Desafio

Neste desafio, foi proposto:

* Construir a interface de listagem de vídeos do YouTube
* Aplicar conceitos de **CSS Grid**
* Trabalhar com **Flexbox**
* Criar um layout **responsivo**
* Organizar o código de forma semântica e escalável

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* CSS Grid Layout
* Flexbox
* Google Fonts (Roboto)

---

## 📱 Responsividade

O projeto foi desenvolvido com foco em diferentes tamanhos de tela:

* 💻 Desktop
* 📱 Tablet
* 📲 Mobile

### Comportamento:

* Grid adaptável com `auto-fit` e `minmax`
* Sidebar oculta em telas menores
* Layout reorganizado para melhor usabilidade

---

## 🎨 Layout

A interface contém:

* 🔝 Header com menu, logo e busca
* 📚 Sidebar com navegação
* 🎞️ Grid de vídeos (cards)
* 👤 Avatar e informações do canal
